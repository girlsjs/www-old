# Girls.js webpage

Based on [hexo](http://https://hexo.io/).

## Running locally

1. Clone repo
1. Install hexo (`yarn install`)
1. Go to page directory (`cd page`)
1. Install additional dependencies (`yarn install`)
1. Run webpage (`yarn hexo server`)
